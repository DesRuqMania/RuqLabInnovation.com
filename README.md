## 🚀 How to Use

### Option 1: Direct File Access
Simply open `index.html` in your web browser. No server required for basic functionality.

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python3 -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if installed)
npx http-server
```

Then open `http://localhost:8000` in your browser.

### Option 3: Deploy to Web Hosting
Upload all files to your web hosting provider:
1. Upload `index.html`, `styles.css`, `script.js`, and `logo.png`
2. Ensure all files are in the same directory
3. Access via your domain

## 🔧 Customization

### Change Colors
Edit the color values in `styles.css`:
- Primary Green: `#1DD1A1`
- Dark Text: `#0F0F0F`
- Light Background: `#ffffff`

### Update Content
Edit text directly in `index.html`:
- Headlines and descriptions
- Service information
- Contact details
- Footer content

### Modify Fonts
Update the Google Fonts link in `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@400;600;700&display=swap" rel="stylesheet">
```

## 💡 Tips

1. **Logo**: Replace `logo.png` with your own logo (keep the same filename)
2. **Favicon**: Add a favicon by adding this line to the `<head>`:
   ```html
   <link rel="icon" type="image/png" href="favicon.png">
   ```
3. **Analytics**: Add Google Analytics or other tracking by adding scripts before `</body>`
4. **Email Form**: The contact form currently shows a success message. To actually send emails, you'll need to integrate with a backend service like Formspree or EmailJS
